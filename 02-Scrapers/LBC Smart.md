# LBC Smart

Scrape Leboncoin pour collecter les annonces hardware.

## Points clés
- Source DB : `leboncoin` (pas `lbc`)
- LBC sold proxy = annonces disparues dans [[Monitor LBC]] (<30j listing age, <90j data age)
- Colonne prix : `current_price` (pas `price`)
