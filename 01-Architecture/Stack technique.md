# Stack technique

## Backend
- **Framework :** FastAPI + PostgreSQL
- **Infra :** Debian VPS, Docker Compose (`monark-api` + `monark-postgres`)
- **DB :** `monark_db`, user `monark_user`
- **Projet :** `~/monark_api/`

## Frontend
- **Framework :** React / TypeScript
- **Déploiement :** [[Lovable workflow]] → `monark-market.fr`
- **API :** `api.monark-market.fr`
- **GitHub :** `Arkaniix/monark-market`
